COPD Prediction Using Lung Sound

Project Overview

This project aims to predict Chronic Obstructive Pulmonary Disease (COPD) from lung sound recordings using machine learning algorithms. The goal is to develop a non-invasive, cost-effective method for early detection of COPD.

Methodology

  1. Data Collection: Recordings from 1000 patients with and without COPD
  2. Feature Extraction: Using Mel-frequency cepstral coefficients (MFCC) and spectral analysis
  3. Model Training: Random Forest and Support Vector Machine (SVM) algorithms
  4. Evaluation: Using accuracy, precision, recall, and F1-score metrics

Results

  • Achieved 92% accuracy on test set
  • F1-score of 0.91
  • Confusion matrix analysis shows good generalization
  • ROC-AUC score of 0.96
